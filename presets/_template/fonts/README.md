# fonts/ — preset UI font

Drop one `.woff2` (or `.woff`/`.ttf`) here and point `preset.json` → `font.url` at it
(e.g. `"fonts/YourFont.woff2"`), with `font.family` set to the font's family name.

**License check (required before selling):** only bundle fonts whose license allows
redistribution and commercial use — SIL Open Font License (most Google Fonts) is
safe. Do NOT bundle fonts from Adobe Fonts, MyFonts, or system fonts you don't have
redistribution rights to. Ship the font's license file alongside it.
